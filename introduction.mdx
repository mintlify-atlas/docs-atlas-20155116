# TaskForge API

TaskForge is a production-ready RESTful API for task management. It provides a complete backend solution with JWT authentication, role-based access control (RBAC), advanced task filtering, tag organization, and data export — all deployed on Azure App Service with CI/CD via GitHub Actions.

## Tech stack

| Layer | Technology |
|---|---|
| Framework | Flask 3.1.2 |
| ORM | SQLAlchemy 2.0.44 |
| Authentication | Flask-JWT-Extended 4.7.1 |
| Database (prod) | Azure SQL Database |
| Database (dev) | SQLite |
| Rate limiting | Flask-Limiter 4.0.0 |
| API docs | Flasgger / Swagger UI |
| Testing | pytest 9.0.1 (268 tests, >73% coverage) |
| Deployment | Azure App Service + GitHub Actions |

## API base URL

```
Production:  https://task-forge-gbd6h8gtg8hchve9.chilecentral-01.azurewebsites.net/api
Local:       http://localhost:5000/api
```

All endpoints are prefixed with `/api`. Authentication uses the `Authorization: Bearer <token>` header.

## Interactive documentation

The API includes an embedded Swagger UI at `/api/docs`. You can explore and test all endpoints directly in the browser without any additional tooling.

<Note>
  The live Swagger UI is available at the [production URL](https://task-forge-gbd6h8gtg8hchve9.chilecentral-01.azurewebsites.net/api/docs). All endpoints require a valid JWT token — use `/api/auth/login` first to obtain one.
</Note>
