---
title: Change password
description: Update the current user's password.
---

<Note>
This endpoint is rate-limited to **3 requests per hour** per IP address.
</Note>

`POST /api/auth/change-password`

**Authentication required.** Pass your access token in the `Authorization: Bearer` header.

## Request body

<ParamField body="old_password" type="string" required>
  The user's current password. The request is rejected if this does not match the stored password hash.
</ParamField>

<ParamField body="new_password" type="string" required>
  The new password. Must satisfy all password requirements (see below).
</ParamField>

## Password requirements

All passwords — including `new_password` — must meet the following criteria:

- **Minimum 8 characters**, maximum 128 characters
- At least one **uppercase letter** (A–Z)
- At least one **lowercase letter** (a–z)
- At least one **number** (0–9)

<Warning>
Changing your password immediately revokes **all active refresh tokens** for your account. Any device using an existing refresh token will need to log in again.
</Warning>

## Response

### 200 OK

The password was changed successfully.

<ResponseField name="success" type="boolean" required>
  Always `true` on success.
</ResponseField>

<ResponseField name="message" type="string">
  Confirmation message.
</ResponseField>

### Error responses

| Status | Cause |
|---|---|
| `400` | Request body is missing or `old_password`/`new_password` is absent |
| `400` | `new_password` does not meet password requirements |
| `401` | `old_password` does not match the current password |
| `401` | Access token is missing, expired, or invalid |

<CodeGroup>

```bash curl
curl --request POST \
  --url http://localhost:5000/api/auth/change-password \
  --header 'Authorization: Bearer <your_access_token>' \
  --header 'Content-Type: application/json' \
  --data '{
    "old_password": "SecurePass123",
    "new_password": "NewSecurePass456"
  }'
```

```python python
import requests

response = requests.post(
    "http://localhost:5000/api/auth/change-password",
    headers={"Authorization": "Bearer <your_access_token>"},
    json={
        "old_password": "SecurePass123",
        "new_password": "NewSecurePass456",
    },
)
print(response.json())
```

</CodeGroup>

**200 response:**

```json
{
  "success": true,
  "message": "Contrasena cambiada con exito"
}
```

**401 response (wrong current password):**

```json
{
  "success": false,
  "message": "La contrasena actual es incorrecta"
}
```
