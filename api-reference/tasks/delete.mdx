---
title: Delete a task
description: Permanently delete a task by its ID.
---

Permanently deletes a task. This action cannot be undone.

You can delete a task you own. Admins can delete any task.

<Info>
  **Authentication required.** Include a valid Bearer token in the `Authorization` header.
</Info>

<Danger>
  Deleting a task is permanent and cannot be reversed.
</Danger>

## Path parameters

<ParamField path="task_id" type="integer" required>
  The ID of the task to delete.
</ParamField>

## Response

Returns a success message with HTTP status `200`.

<ResponseField name="success" type="boolean" required>
  `true` on a successful deletion.
</ResponseField>

<ResponseField name="message" type="string">
  Confirmation message.
</ResponseField>

## Errors

| Status | Description |
|--------|-------------|
| `403`  | You do not own this task and are not an admin. |
| `404`  | No task with the given `task_id` exists. |

<RequestExample>

```bash cURL
curl --request DELETE \
  --url 'https://api.example.com/api/tasks/42' \
  --header 'Authorization: Bearer <token>'
```

```python Python
import requests

response = requests.delete(
    "https://api.example.com/api/tasks/42",
    headers={"Authorization": "Bearer <token>"},
)
print(response.json())
```

</RequestExample>

<ResponseExample>

```json 200
{
  "success": true,
  "message": "Tarea eliminada con exito"
}
```

```json 403
{
  "success": false,
  "message": "No tienes permiso para eliminar esta tarea"
}
```

```json 404
{
  "success": false,
  "message": "Tarea no encontrada"
}
```

</ResponseExample>
