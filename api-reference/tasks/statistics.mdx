---
title: Get task statistics
description: Retrieve aggregated task counts and breakdowns for the authenticated user.
---

Returns task statistics for the authenticated user, including counts by status, counts by priority, and the number of overdue tasks.

<Info>
  **Authentication required.** Include a valid Bearer token in the `Authorization` header.
</Info>

<Note>
  Statistics reflect only the tasks owned by the authenticated user. Admins receive statistics scoped to their own account, not all users.
</Note>

## Response

<ResponseField name="success" type="boolean" required>
  `true` on a successful response.
</ResponseField>

<ResponseField name="data" type="object" required>
  <Expandable title="properties" defaultOpen>
    <ResponseField name="total_tasks" type="integer">
      Total number of tasks owned by the user.
    </ResponseField>
    <ResponseField name="completed_tasks" type="integer">
      Number of tasks with status `completed`.
    </ResponseField>
    <ResponseField name="pending_tasks" type="integer">
      Number of tasks with status `pending`.
    </ResponseField>
    <ResponseField name="in_progress_tasks" type="integer">
      Number of tasks with status `in_progress`.
    </ResponseField>
    <ResponseField name="cancelled_tasks" type="integer">
      Number of tasks with status `cancelled`.
    </ResponseField>
    <ResponseField name="overdue_tasks" type="integer">
      Number of tasks whose `due_date` is in the past and status is not `completed`.
    </ResponseField>
    <ResponseField name="by_priority" type="object">
      Task counts broken down by priority level.
      <Expandable title="properties">
        <ResponseField name="low" type="integer">
          Number of tasks with priority `low`.
        </ResponseField>
        <ResponseField name="medium" type="integer">
          Number of tasks with priority `medium`.
        </ResponseField>
        <ResponseField name="high" type="integer">
          Number of tasks with priority `high`.
        </ResponseField>
        <ResponseField name="urgent" type="integer">
          Number of tasks with priority `urgent`.
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

## Errors

| Status | Description |
|--------|-------------|
| `404`  | The authenticated user was not found. |

<RequestExample>

```bash cURL
curl --request GET \
  --url 'https://api.example.com/api/tasks/statistics' \
  --header 'Authorization: Bearer <token>'
```

```python Python
import requests

response = requests.get(
    "https://api.example.com/api/tasks/statistics",
    headers={"Authorization": "Bearer <token>"},
)
print(response.json())
```

</RequestExample>
