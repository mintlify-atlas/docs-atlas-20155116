---
title: Get a user
description: Retrieve the profile of a single user by their ID.
---

Returns the profile of a single user. Regular users can only retrieve their own profile; admins can retrieve any user's profile.

## Authentication

Requires a valid Bearer token in the `Authorization` header.

<Info>
  A non-admin user requesting a `user_id` that is not their own will receive `403 Forbidden`.
</Info>

## Path parameters

<ParamField path="user_id" type="number" required>
  The unique integer ID of the user to retrieve.
</ParamField>

## Response fields

<ResponseField name="success" type="boolean" required>
  `true` when the request succeeds.
</ResponseField>

<ResponseField name="data" type="object" required>
  <Expandable title="properties" defaultOpen={true}>
    <ResponseField name="id" type="number" required>
      Unique identifier for the user.
    </ResponseField>
    <ResponseField name="username" type="string" required>
      The user's unique username.
    </ResponseField>
    <ResponseField name="email" type="string" required>
      The user's email address.
    </ResponseField>
    <ResponseField name="first_name" type="string">
      User's first name. `null` if not set.
    </ResponseField>
    <ResponseField name="last_name" type="string">
      User's last name. `null` if not set.
    </ResponseField>
    <ResponseField name="full_name" type="string" required>
      Computed display name. Returns `first_name last_name` when both are set, otherwise falls back to `username`.
    </ResponseField>
    <ResponseField name="is_active" type="boolean" required>
      Whether the user can authenticate.
    </ResponseField>
    <ResponseField name="role" type="object" required>
      The user's assigned role.
      <Expandable title="role properties">
        <ResponseField name="id" type="number" required>
          Role ID.
        </ResponseField>
        <ResponseField name="name" type="string" required>
          Role name: `admin` or `user`.
        </ResponseField>
        <ResponseField name="description" type="string">
          Role description.
        </ResponseField>
      </Expandable>
    </ResponseField>
    <ResponseField name="created_at" type="string" required>
      ISO 8601 timestamp of account creation.
    </ResponseField>
    <ResponseField name="updated_at" type="string" required>
      ISO 8601 timestamp of last profile update.
    </ResponseField>
  </Expandable>
</ResponseField>

## Errors

| Status | Description |
| ------ | ----------- |
| `401`  | Missing or invalid Bearer token. |
| `403`  | Requesting another user's profile without admin privileges. |
| `404`  | No user exists with the given `user_id`. |

## Examples

<CodeGroup>

```bash cURL
curl --request GET \
  --url https://api.taskforge.example.com/api/users/2 \
  --header 'Authorization: Bearer <token>'
```

```python Python
import requests

response = requests.get(
    "https://api.taskforge.example.com/api/users/2",
    headers={"Authorization": "Bearer <token>"},
)
print(response.json())
```

</CodeGroup>
