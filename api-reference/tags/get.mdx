---
title: Get a tag
description: Retrieve a single tag by its ID.
---

Returns the details of a single tag. Any authenticated user can call this endpoint.

## Authentication

Requires a valid Bearer token in the `Authorization` header.

## Path parameters

<ParamField path="tag_id" type="number" required>
  The unique integer ID of the tag to retrieve.
</ParamField>

## Response fields

<ResponseField name="success" type="boolean" required>
  `true` when the request succeeds.
</ResponseField>

<ResponseField name="data" type="object" required>
  <Expandable title="properties" defaultOpen={true}>
    <ResponseField name="id" type="number" required>
      Unique identifier for the tag.
    </ResponseField>
    <ResponseField name="name" type="string" required>
      Tag name.
    </ResponseField>
    <ResponseField name="color" type="string" required>
      Hex color code (e.g. `#FF5733`).
    </ResponseField>
    <ResponseField name="description" type="string">
      Optional description of the tag. `null` if not set.
    </ResponseField>
    <ResponseField name="task_count" type="number" required>
      Number of tasks currently assigned this tag.
    </ResponseField>
    <ResponseField name="created_at" type="string" required>
      ISO 8601 timestamp of when the tag was created.
    </ResponseField>
    <ResponseField name="updated_at" type="string" required>
      ISO 8601 timestamp of when the tag was last updated.
    </ResponseField>
  </Expandable>
</ResponseField>

## Errors

| Status | Description |
| ------ | ----------- |
| `401`  | Missing or invalid Bearer token. |
| `404`  | No tag exists with the given `tag_id`. |

## Examples

<CodeGroup>

```bash cURL
curl --request GET \
  --url https://api.taskforge.example.com/api/tags/1 \
  --header 'Authorization: Bearer <token>'
```

```python Python
import requests

response = requests.get(
    "https://api.taskforge.example.com/api/tags/1",
    headers={"Authorization": "Bearer <token>"},
)
print(response.json())
```

</CodeGroup>
