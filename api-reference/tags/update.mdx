---
title: Update a tag
description: Update the name, color, or description of an existing tag.
---

Updates one or more fields of an existing tag. Supports both full (`PUT`) and partial (`PATCH`) updates — only fields you include in the request body are modified.

<Warning>
  This endpoint requires **admin** privileges. Requests from non-admin users return `403 Forbidden`.
</Warning>

## Authentication

Requires a valid Bearer token from an admin account in the `Authorization` header.

## Path parameters

<ParamField path="tag_id" type="number" required>
  The unique integer ID of the tag to update.
</ParamField>

## Request body

All fields are optional. At least one field must be provided.

<ParamField body="name" type="string">
  New name for the tag. Must be between 1 and 50 characters and unique across all tags.
</ParamField>

<ParamField body="color" type="string">
  New hex color code (e.g. `#FF5733`). Must match the pattern `^#[0-9A-Fa-f]{6}$`.
</ParamField>

<ParamField body="description" type="string">
  New description for the tag. Pass `null` or an empty string to clear it.
</ParamField>

## Response fields

<ResponseField name="success" type="boolean" required>
  `true` when the update succeeds.
</ResponseField>

<ResponseField name="message" type="string" required>
  Confirmation message.
</ResponseField>

<ResponseField name="data" type="object" required>
  The updated tag object.
  <Expandable title="properties" defaultOpen={true}>
    <ResponseField name="id" type="number" required>
      Unique identifier of the tag.
    </ResponseField>
    <ResponseField name="name" type="string" required>
      Tag name.
    </ResponseField>
    <ResponseField name="color" type="string" required>
      Hex color code.
    </ResponseField>
    <ResponseField name="description" type="string">
      Tag description. `null` if not set.
    </ResponseField>
    <ResponseField name="task_count" type="number" required>
      Number of tasks assigned this tag.
    </ResponseField>
    <ResponseField name="created_at" type="string" required>
      ISO 8601 timestamp of creation.
    </ResponseField>
    <ResponseField name="updated_at" type="string" required>
      ISO 8601 timestamp of the most recent update.
    </ResponseField>
  </Expandable>
</ResponseField>

## Errors

| Status | Description |
| ------ | ----------- |
| `400`  | Validation failed — `name` is too long or `color` is not a valid hex code. |
| `401`  | Missing or invalid Bearer token. |
| `403`  | Authenticated user is not an admin. |
| `404`  | No tag exists with the given `tag_id`. |
| `409`  | A different tag with the requested `name` already exists. |

## Examples

<CodeGroup>

```bash cURL
curl --request PATCH \
  --url https://api.taskforge.example.com/api/tags/5 \
  --header 'Authorization: Bearer <admin-token>' \
  --header 'Content-Type: application/json' \
  --data '{
    "color": "#EF4444",
    "description": "Critical items requiring same-day resolution"
  }'
```

```python Python
import requests

response = requests.patch(
    "https://api.taskforge.example.com/api/tags/5",
    headers={
        "Authorization": "Bearer <admin-token>",
        "Content-Type": "application/json",
    },
    json={
        "color": "#EF4444",
        "description": "Critical items requiring same-day resolution",
    },
)
print(response.json())
```

</CodeGroup>
